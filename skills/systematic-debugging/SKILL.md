# Systematic Debugging

This skill provides a structured approach to debugging issues.

## Capabilities

- Root cause analysis
- Reproduction steps creation
- Debugging workflow management
- Stack trace interpretation

## When Active

This skill is automatically loaded when user prompts mention:
- debug, trace, repro, bug, error, fix, issue

## Guidelines

1. Reproduce the issue consistently first
2. Isolate the problem area
3. Form and test hypotheses
4. Document findings as you go
5. Verify the fix doesn't introduce regressions
