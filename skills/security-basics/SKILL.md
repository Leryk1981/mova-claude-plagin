# Security Basics

This skill provides guidance on secure coding practices.

## Capabilities

- Input validation and sanitization
- Authentication and authorization
- Common vulnerability prevention (XSS, CSRF, injection)
- Secure configuration management

## When Active

This skill is automatically loaded when user prompts mention:
- security, vuln, xss, csrf, auth, injection, sanitize

## Guidelines

1. Never trust user input
2. Use parameterized queries
3. Implement proper authentication
4. Follow principle of least privilege
5. Keep secrets out of code
6. Log security events appropriately
