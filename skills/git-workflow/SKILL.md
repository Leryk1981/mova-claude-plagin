# Git Workflow

This skill provides guidance on Git operations and version control.

## Capabilities

- Branch management strategies
- Commit message conventions
- Merge and rebase workflows
- Conflict resolution

## When Active

This skill is automatically loaded when user prompts mention:
- git, branch, merge, rebase, commit, push, pull

## Guidelines

1. Use feature branches for development
2. Write clear, descriptive commit messages
3. Keep commits atomic and focused
4. Never force push to shared branches
5. Resolve conflicts carefully
6. Review changes before committing
